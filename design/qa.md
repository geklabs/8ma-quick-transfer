# Design QA

prototype: https://t.8ma.co/about/
source: ./design/context.md + ../DESIGN.md

## Visual

- ✓ 中文桌面 1440px：非对称首屏、8位码视觉锚点、编辑式内容节奏符合设计规范
- ✓ 英文桌面 1440px：长标题与事实内容无溢出，层级与中文版一致
- ✓ 中文与英文移动端 375×812：主 CTA 位于首屏，页面无横向溢出
- ✓ 生产环境标题比例已复核：桌面 H1 50px、H2 40px；375px 移动端 H1 35.625px、H2 28px
- ✓ 中英文标题均无横向溢出；中文首屏标题在桌面和 375px 移动端保持一行
- ✓ 中文桌面主要段落标题保持一行，连接边界长标题保留自然两行
- ✓ 品牌蓝、暖白、深墨与橙色状态色符合设计令牌
- ✓ 未使用装饰性渐变、玻璃效果、渐变文字或同构功能卡片

## Behavior

- ✓ 浏览器首选语言自动选择中文或英文
- ✓ 手动语言切换写入本地偏好，并在刷新后保持
- ✓ 隐私与使用说明保持对应语言路径
- ✓ FAQ 支持键盘 Enter 展开
- ✓ 所有主要 CTA 指向 https://t.8ma.co
- ✓ 减少动态效果偏好受到尊重

## Requirements

- ✓ 独立公开推广内容，不包含产品源码、服务器配置、凭据或内部数据
- ✓ 生产环境页面已完成验证
- ✓ 中英文拥有独立、可索引 URL
- ✓ canonical、hreflang、Open Graph、Twitter Card 完整
- ✓ SoftwareApplication、WebSite、FAQPage 结构化数据可解析
- ✓ sitemap.xml、robots.txt、llms.txt 与 manifest 可解析
- ✓ 产品能力与网络条件边界表述准确，没有承诺所有环境下都能完成传输
- ✓ 404、隐私说明、使用说明和返回路径齐全
- ✓ 未知推广路径返回真实 HTTP 404，而不是软 404

## Accessibility

- ✓ 跳到主要内容链接是首个键盘焦点
- ✓ 页头可见交互目标至少 40×40px
- ✓ 可见焦点样式明确
- ✓ 语义标题、导航、列表、定义表与 details/summary 均可访问
- manual: 完整屏幕阅读器走查仍需真实辅助技术复核

## SEO / GEO

- ✓ 服务端静态 HTML 中包含完整核心内容，不依赖 JavaScript 注入
- ✓ 两种语言具有不同标题、描述与明确搜索意图
- ✓ 产品事实、边界说明和问答采用可提取的语义结构
- ✓ 公开内容只描述用户效果，不包含协议、网络架构、连接服务或算法名称
- ✓ 社交分享图为 1200×630 PNG
- ✓ 无分析脚本、广告脚本或 Cookie 横幅需求
