免登录传文件

不用注册登录，也能直接传文件。

8码快传不要求发送方或接收方创建账号。打开网页、选择文件，再把接收方式交给对方，几步就能开始，而且目前免费使用。

直接答案

打开 8码快传选择文件，同一 WiFi 下使用附近分享，也可以复制链接、展示二维码或告诉对方 8 位提取码。对方打开网站后确认接收即可。

三步免登录传文件

  1. 发送方打开 8码快传，选择要发送的文件。
  2. 选择附近分享、链接、二维码或 8 位提取码。
  3. 接收方打开网站并确认下载，双方保持页面打开。

为什么不需要账号？

一次传输只需要临时的接收方式。接收方不用添加好友、绑定手机号或登录同一个账号，也不用安装新的应用。

文件会被长期保存吗？

不会生成可长期下载的文件副本。发送和接收页面需要在传输期间保持打开，结束后分享随会话失效。

免费使用有什么限制？

8码快传目前免费使用。桌面版 Chrome 和 Edge 单文件最高支持 100 GB，实际传输速度和可用性取决于设备、浏览器和网络条件。
