同 WiFi 互传

电脑和手机打开网站，就能互传文件。

同一 WiFi 下，不管是电脑传电脑，还是电脑传手机，都不需要安装应用或登录账号。两台设备打开 8码快传，发送方开启附近分享，接收方就能看到文件。

最简单的操作

发送方选择文件并开启“同 WiFi 共享”，接收方在另一台设备打开网站，点击附近出现的文件即可接收。

电脑传电脑

  1. 两台电脑连接同一 WiFi，并分别打开 8码快传。
  2. 发送方选择文件，开启“同 WiFi 共享”。
  3. 接收方在“附近的文件”中打开分享并确认下载。

电脑传手机

操作完全一样。电脑选好文件并开启附近分享，手机打开网站后即可看到。反过来从手机传到电脑，也不需要安装额外应用。

为什么方便？

  • 不需要复制长链接，也不需要登录同一个账号。
  • Windows、macOS、Android 和 iPhone 都可以直接使用浏览器。
  • 文件选好就能开始传，速度和剩余时间随时可见。

公共 WiFi 可以用吗？

可以使用链接、二维码或 8 位提取码，但不建议在公共网络开启附近分享，以免其他同网络设备看到分享所展示的文件名。
