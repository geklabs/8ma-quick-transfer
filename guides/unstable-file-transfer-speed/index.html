传输速度

文件传输速度为什么不稳定，怎么提高速度？

一次传输的速度，由发送设备、接收设备和当前可用网络中最慢的一环决定。任何一项条件发生变化，页面显示的速度都可能跟着变化。

直接答案

尽量使用同一 WiFi，让两台设备靠近信号较强的接入点，暂停其他下载，避免设备休眠，保持双方页面打开，并使用最新版 Chrome 或 Edge。追求最高速度时，一次只连接一个接收者。

先分清 Mbps 和 MB/s

网络速率通常使用兆比特每秒（Mbps），文件传输速度通常使用兆字节每秒（MB/s）。8 比特等于 1 字节，所以 100 Mbps 在没有其他损耗前相当于 12.5 MB/s，并不代表能够达到 100 MB/s。

让大文件传输更快、更稳定的 8 个方法

  1. 尽量使用同一局域网络。两台设备同时打开 8码快传，通过附近分享接收，避免跨不同网络传输。
  2. 改善 WiFi 信号。让两台设备靠近接入点，避开信号较弱的房间和拥挤的公共 WiFi。
  3. 暂停其他大流量任务。在线视频、云备份、游戏下载和其他文件传输都会占用当前可用带宽。
  4. 保持两台设备唤醒。超大文件传输时可以接通电源，避免省电模式暂停浏览器。
  5. 保持双方页面打开。传输期间不要关闭、刷新、后退或跳转页面。
  6. 使用最新版浏览器。超大文件推荐桌面版 Chrome 或 Edge，单文件最高支持 100 GB。
  7. 同时检查两台设备。即使网络很快，设备繁忙或本地读写较慢也可能成为整次传输的速度上限。
  8. 追求最高速度时只连接一个接收者。一次分享最多支持 3 人接收，但多个接收者会共同使用发送设备当前可用的资源。

为什么刚开始很快，后来速度下降？

刚开始的短时间速度可能使用了暂时空闲的带宽。其他流量开始、WiFi 信号变化、设备进入省电状态，或者本地读写成为最慢环节，速度都会下降。判断实际效果应看整次传输的持续平均速度，而不是某一瞬间的最高值。

传输中断了怎么办？

在支持的浏览器中，已经完成的部分可以保留在接收设备本地。重新打开分享后，发送方再次确认同一个原文件，即可从校验通过的断点继续，不必整份文件重新开始。
