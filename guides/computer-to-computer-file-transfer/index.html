电脑互传

两台电脑怎么快速互传文件？

没有 U 盘、数据线或共享文件夹，也可以让两台电脑直接开始传文件。双方只需打开浏览器，不用安装应用，也不用注册账号。

直接答案

两台电脑打开 8码快传。发送方选择文件；同一 WiFi 下开启附近分享，接收方就能看到文件。不在同一网络时，发送链接或 8 位提取码即可。

同一 WiFi 下怎么传？

  1. 两台电脑都打开 8码快传。
  2. 发送方选择文件并开启“同 WiFi 共享”。
  3. 接收方在“附近的文件”中选择文件并确认下载。

不在同一网络怎么办？

发送方可以复制临时链接、展示二维码或发送 8 位提取码。接收方打开链接或输入提取码后即可下载。

Windows 和 Mac 能互传吗？

可以。只要两台电脑能使用现代浏览器，就可以在 Windows、macOS 或 Linux 之间传文件。大文件推荐使用最新版 Chrome 或 Edge。

适合传大文件吗？

适合。文件不用先上传网盘，会充分利用当前网络的可用带宽；桌面 Chrome 和 Edge 单文件最高支持 100 GB。
