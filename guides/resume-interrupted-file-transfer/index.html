断点续传

大文件传输中断后怎么继续？

网络波动、刷新页面或临时关闭浏览器，不一定要从头再传。8码快传会在支持的浏览器中保留接收设备已经写入的部分，重新连接后从校验通过的断点继续。

直接答案

在原发送设备选择同一个文件继续分享，接收设备重新打开原链接并点击“继续接收”。系统会先校验已保存部分，再从断点继续传输。

中断后怎样恢复？

  1. 发送方回到 8码快传，选择“继续上次分享”，并重新选择原文件。
  2. 接收方重新打开原来的分享链接或 8 位提取码页面。
  3. 点击“继续接收”，等待已保存部分校验完成后继续传输。

哪些内容不会重复传？

接收设备已经完成校验并保存到本地的部分不会重新发送。若本地内容与原文件不一致，系统会放弃错误断点并重新开始，避免得到损坏文件。

什么时候需要重新选择原文件？

发送页面关闭或刷新后，浏览器不能擅自读取本地文件。发送方需要重新确认原文件，文件名、大小和修改时间匹配后才能继续分享。

怎样减少再次中断？

  • 传输期间保持双方页面打开，不要让设备休眠。
  • 大文件优先使用最新版桌面 Chrome 或 Edge。
  • 尽量使用稳定网络；同一 WiFi 下可直接开启附近分享。
