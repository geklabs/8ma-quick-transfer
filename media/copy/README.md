- 可以写充分利用当前网络的可用带宽，实际速度取决于网络和设备。
- 100 GB 需要限定为桌面版 Chrome 或 Edge 的单文件上限。
- 不介绍连接协议、网络穿透、信令或其他实现细节。
- 不承诺所有网络都能连接，不写“永久有效”或“绝对安全”。
