# Product Hunt

## Name

8ma Quick Transfer

## Tagline

Fast file transfers with no upload wait or sign-up

## Description

8ma Quick Transfer lets you send a file from one device to another in the browser. Pick a file, then share a temporary link, QR code, or 8-character pickup code. On the same WiFi, open the site on both devices and enable nearby sharing.

It is free to use, requires no account, and does not ask the receiver to install an app. The transfer uses the available bandwidth of the current network. Actual speed depends on both devices and their network. Desktop Chrome and Edge support files up to 100 GB.

Product URL:

https://t.8ma.co/?from=producthunt

## First comment

We built 8ma Quick Transfer for a common interruption: a large file is on one device, but the person or device that needs it is somewhere else.

The sender chooses a file and gets three ways to hand it off: a link, a QR code, or an 8-character pickup code. If both devices use the same WiFi, they can open the site and discover the share without copying a link.

The current product is free and has no account flow. Both pages need to stay open during the transfer, and performance depends on the devices and network. We would value feedback on phone-to-computer transfers, browser compatibility, and the clarity of the receiving flow.
