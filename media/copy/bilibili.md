# Bilibili

## 视频标题

不用注册和安装，电脑手机怎样快速互传大文件

## 简介

8码快传是一个免费的浏览器文件传输工具。发送方选择文件后，可以通过链接、二维码或 8 位提取码让另一台设备接收。

同一 WiFi 下，电脑和手机同时打开网站，开启附近分享即可互传。传输会充分利用当前网络的可用带宽，实际速度取决于设备和网络。桌面版 Chrome 或 Edge 单文件最高支持 100 GB。

立即使用：https://t.8ma.co/?from=bilibili

使用指南：https://t.8ma.co/about/guides/?from=bilibili

4. 同一 WiFi，双方打开网站即可互传
5. 免费使用，无需注册登录
