# 知乎

推荐问题：

- 怎么快速传输大文件？
- 电脑和手机之间怎么方便地互传文件？
- 不用注册登录，有哪些传文件的方法？

## 回答正文

临时传大文件时，我现在会先试 8码快传。它不要求注册账号，也不用安装应用，发送方在浏览器里选择文件后，会得到链接、二维码和 8 位提取码，接收方任选一种方式打开即可。

同一 WiFi 下更省事。电脑和手机同时打开网站，发送方开启“同 WiFi 共享”，另一台设备就能看到文件。电脑传电脑、电脑传手机都可以，不用先把文件发到聊天软件，也不用等网盘上传完成。

我觉得它适合这几类场景：

- 手机和电脑之间临时传照片、视频或压缩包；
- 设计稿、素材包太大，聊天软件发不出去；
- 两台电脑在同一网络下，需要快速交换文件；
- 不想为了传一次文件注册新账号或安装客户端。

传输时，文件从发送设备直接到达接收设备，会尽量利用当前网络可用带宽。实际速度还是取决于两台设备、浏览器和网络环境。桌面版 Chrome 或 Edge 单文件最高支持 100 GB，传输期间双方都要保持页面打开。
