Une ligne

8ma Quick Transfer est un outil gratuit de transfert de fichiers par navigateur, sans inscription, utilisant des liens temporaires, des codes QR, des codes de retrait à 8 caractères et le partage sur le même réseau WiFi.

Description courte

8ma Quick Transfer aide les utilisateurs à transférer des fichiers entre téléphones et ordinateurs. Choisissez un fichier, puis partagez un lien temporaire, un code QR ou un code de retrait à 8 caractères. Sur le même réseau WiFi, ouvrez le site sur les deux appareils pour découvrir le partage. C'est gratuit et ne nécessite ni compte ni installation d'application.

Faits vérifiés
  • Gratuit, sans inscription ni installation d'application
  • Lien temporaire, code QR et code de retrait à 8 caractères
  • Partage sur le même réseau WiFi entre téléphones et ordinateurs
  • Jusqu'à trois destinataires par partage
  • Jusqu'à 100 Go par fichier sur Chrome ou Edge pour ordinateur
  • La vitesse réelle dépend des appareils et du réseau ; les deux pages doivent rester ouvertes
