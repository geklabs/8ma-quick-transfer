Transfert sur le même WiFi

Ouvrez le site sur les deux appareils et commencez à partager.

Sur le même WiFi, les transferts d'ordinateur à ordinateur et d'ordinateur à téléphone ne nécessitent aucune application supplémentaire ni compte partagé. Ouvrez 8ma Quick Transfer sur les deux appareils, puis activez le Partage à proximité sur l'émetteur.

Le chemin le plus simple

L'émetteur choisit un fichier et active « Partage sur le même WiFi ». Le destinataire ouvre le site sur l'autre appareil et sélectionne le fichier sous Fichiers à proximité.

D'ordinateur à ordinateur

  1. Connectez les deux ordinateurs au même WiFi et ouvrez 8ma Quick Transfer.
  2. Choisissez le fichier sur l'émetteur et activez le partage sur le même WiFi.
  3. Ouvrez le fichier à proximité sur l'ordinateur destinataire et confirmez le téléchargement.

D'ordinateur à téléphone

Les étapes sont les mêmes. Choisissez le fichier sur l'ordinateur, activez le Partage à proximité et ouvrez le site sur le téléphone. L'envoi d'un téléphone vers un ordinateur fonctionne de la même manière.

Pourquoi est-ce pratique ?

  • Aucun long lien à copier et aucun compte partagé auquel se connecter.
  • Windows, macOS, Android et iPhone peuvent utiliser un navigateur.
  • La vitesse et le temps restant restent visibles pendant le déplacement du fichier.

Puis-je utiliser un WiFi public ?

Utilisez plutôt un lien, un code QR ou un code de retrait. Gardez le Partage à proximité désactivé sur les réseaux publics afin que les autres appareils ne voient pas le nom de fichier affiché par le partage.
