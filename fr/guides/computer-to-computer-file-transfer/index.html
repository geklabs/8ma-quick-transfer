Transfert entre ordinateurs

Comment transférer des fichiers d'ordinateur à ordinateur ?

Même sans clé USB, câble ou dossier partagé, deux ordinateurs peuvent commencer un transfert via leurs navigateurs. Aucune application à installer ni compte à créer.

La réponse courte

Ouvrez 8ma Quick Transfer sur les deux ordinateurs. Choisissez le fichier sur l'émetteur ; sur le même WiFi, activez le partage de proximité. Sur des réseaux différents, envoyez plutôt un lien ou un code de récupération.

Transférer sur le même WiFi

  1. Ouvrez 8ma Quick Transfer sur les deux ordinateurs.
  2. Choisissez le fichier sur l'émetteur et activez le partage sur le même WiFi.
  3. Sélectionnez le fichier sous Fichiers à proximité sur l'ordinateur récepteur et confirmez le téléchargement.

Que faire si les ordinateurs utilisent des réseaux différents ?

L'émetteur peut copier un lien temporaire, afficher un QR code ou partager le code de récupération à 8 caractères. Le destinataire ouvre le lien ou saisit le code pour télécharger.

Windows et Mac peuvent-ils échanger des fichiers entre eux ?

Oui. Un navigateur récent permet aux ordinateurs Windows, macOS et Linux d'échanger des fichiers. Pour les très gros fichiers, utilisez la version actuelle de Chrome ou Edge pour ordinateur.

Cela fonctionne-t-il pour les fichiers volumineux ?

Oui. Il n'y a pas de téléversement cloud séparé, et le transfert utilise la bande passante réseau disponible. Chrome et Edge pour ordinateur prennent en charge les fichiers jusqu'à 100 Go.
