Caractéristiques du produit

Quand est-ce utile ?

Pour déplacer des photos, vidéos, fichiers de conception et archives entre appareils sans les télécharger d'abord sur un cloud.

Accès
Fonctionne dans le navigateur sans installation d'application ni création de compte
Moyens de partage
Lien temporaire, QR code et code de retrait alphanumérique à 8 caractères
Transfert sur le même WiFi
Ouvrez le site sur un ordinateur et un téléphone, puis activez le Partage de proximité pour découvrir le fichier ; désactivez-le sur les réseaux publics
Destinataires
Jusqu'à trois personnes peuvent recevoir en même temps, avec une progression distincte pour chaque appareil
Gros fichiers
Jusqu'à 100 Go par fichier sur Chrome et Edge pour ordinateur ; les résultats réels dépendent de l'appareil et des conditions réseau
Reprise
Les navigateurs pris en charge peuvent continuer un transfert interrompu à partir des données déjà sauvegardées sur l'appareil destinataire
Configuration recommandée
Utilisez une version actuelle de Chrome ou Edge et gardez les deux pages ouvertes pendant le transfert

Questions

Ce que vous voudrez peut-être savoir d'abord.

Dois-je installer une application ?
Non. Les expéditeurs et les destinataires peuvent ouvrir 8ma Quick Transfer directement dans un navigateur moderne. Une version actuelle de Chrome ou Edge est recommandée.
8ma Quick Transfer conserve-t-il une copie du fichier ?
Non. 8ma Quick Transfer ne conserve pas de copie téléchargeable à long terme. Gardez les deux pages ouvertes pendant le transfert ; le partage se termine à la fin de la session.
Comment envoyer un fichier à un autre appareil ?
Sur le même WiFi, ouvrez le site sur les deux appareils et activez le Partage de proximité sur l'expéditeur. L'appareil destinataire peut alors voir le fichier. Sur différents réseaux, utilisez un lien, un QR code ou un code de retrait à 8 caractères.
Plus d'une personne peut-elle recevoir un fichier ?
Oui. Un partage prend en charge jusqu'à trois destinataires simultanés, et la page d'envoi montre la connexion et la progression du transfert de chaque appareil.
Pourquoi une connexion peut-elle échouer ?
L'état de l'appareil, la version du navigateur et les conditions réseau peuvent affecter un transfert. Gardez la page au premier plan, utilisez la version actuelle de Chrome ou Edge, ou essayez un autre réseau.
Dois-je recommencer après une interruption ?
Pas toujours. Les navigateurs pris en charge conservent la partie terminée sur l'appareil destinataire. Une fois que l'expéditeur confirme à nouveau le fichier d'origine, le destinataire peut continuer à partir d'un point de contrôle vérifié.
