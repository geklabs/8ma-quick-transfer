## 不上传文件，也能发给别人

8码快传是一款浏览器端点对点文件传输工具。选择文件后，页面会生成临时链接、二维码和一组 8 位取件码。接收方打开链接或输入 8 码，即可从发送设备直接接收文件。

文件内容不经过 8码快传服务器。服务器只协助两台设备建立连接，不读取、不保存文件。

## 常用的分享方式都在这里

- **链接、二维码和 8 码**：发链接适合聊天，二维码适合跨设备，8 码适合当面口述。
- **同 WiFi 发现**：临时开启附近共享后，同一网络里的设备可以直接找到文件。
- **多人接收**：一个分享最多支持 3 人同时下载，并分别显示传输进度。
- **大文件直传**：桌面版 Chrome 和 Edge 单文件最高支持 100 GB。
- **传输状态清楚可见**：实时显示速度、剩余时间和接收状态，完成后校验文件完整性。

## 三步完成传输

1. 在发送设备打开 [t.8ma.co](https://t.8ma.co)，选择文件。
2. 把链接、二维码或 8 码交给接收方。
3. 双方保持页面打开，等待传输完成。

## 使用提示

- 推荐使用最新版 Chrome 或 Edge。
- 发送和接收过程中请保持页面打开，关闭或刷新页面会中断传输。
- 点对点连接会受到浏览器、设备和网络环境影响；连接失败时可以更换网络后重试。
- 在公共 WiFi 下谨慎开启“同 WiFi 共享”，避免向同网络的陌生设备展示文件名。
