전송 가이드

이동해야 할 파일부터 시작하세요.

수 기가바이트 파일이든, 컴퓨터와 휴대폰 간 빠른 전송이든, 설치할 앱도 만들 계정도 없습니다. 상황에 가장 맞는 가이드를 선택하세요.

  • 대용량 파일을 빠르게 전송하는 방법
  • 휴대폰과 컴퓨터 간 파일 전송 방법
  • 동일 WiFi에서 파일 전송하는 방법
  • 회원가입 없이 파일 전송하는 방법
  • 메시지 앱으로 보내기 너무 큰 파일 전송 방법
  • 중단된 대용량 파일 전송을 재개하는 방법
  • 클라우드 업로드 없이 파일 전송하는 방법
  • 컴퓨터에서 컴퓨터로 파일 전송하는 방법
  • 파일 전송 속도가 불안정한 이유와 개선 방법
시작할 준비가 되셨나요?
