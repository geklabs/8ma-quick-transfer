대용량 파일 전송

대용량 파일을 어떻게 빠르게 전송하나요?

클라우드 업로드와 2차 다운로드를 기다릴 필요가 없습니다. 8ma Quick Transfer는 파일이 있는 기기에서 바로 전송을 시작하며, 사용 가능한 네트워크 대역폭을 활용하고 앱이나 계정 없이 작동합니다.

간단한 해답

두 기기에서 8ma Quick Transfer를 열고 파일을 선택한 다음, Nearby Sharing, 링크, QR 코드 또는 8자리 픽업 코드를 사용하세요. 전송하는 동안 두 페이지를 모두 열어 두세요.

세 단계로 시작하기

  1. 보내는 기기에서 8ma Quick Transfer를 열고 파일을 선택하세요.
  2. 동일한 WiFi에서 Nearby Sharing을 사용하거나, 링크, QR 코드 또는 픽업 코드를 보내세요.
  3. 받는 사람이 다운로드를 확인하면, 완료될 때까지 두 페이지가 모두 열려 있습니다.

대용량 파일에 왜 유용한가요?

먼저 완료해야 할 별도의 클라우드 업로드가 없습니다. 속도, 남은 시간, 수신자 진행률이 계속 표시됩니다. 데스크톱 Chrome과 Edge는 최대 100GB 파일을 지원합니다.

언제 가장 적합한가요?

  • 고해상도 비디오, 디자인 작업물, 프로젝트 파일 또는 대용량 아카이브를 보낼 때.
  • 다른 앱을 설치하지 않고 컴퓨터 간에 파일을 옮길 때.
  • 컴퓨터와 휴대폰 간에 사진과 동영상을 보낼 때.

먼저 무엇을 확인해야 하나요?

실제 속도는 두 기기와 사용 가능한 네트워크 대역폭에 따라 달라집니다. 매우 큰 파일의 경우 최신 데스크톱 Chrome 또는 Edge를 사용하고 두 기기가 절전 모드로 전환되지 않도록 유지하세요.
