동일 WiFi 전송

두 기기에서 사이트를 열고 공유를 시작하세요.

동일한 WiFi에서는 컴퓨터 간 또는 컴퓨터와 휴대폰 간 전송에 추가 앱이나 공유 계정이 필요하지 않습니다. 두 기기에서 8ma Quick Transfer를 연 다음, 보내는 쪽에서 Nearby Sharing을 활성화하세요.

가장 간단한 방법

보내는 사람이 파일을 선택하고 “동일 WiFi 공유”를 켭니다. 받는 사람은 다른 기기에서 사이트를 열고 Nearby Files 아래에서 파일을 선택합니다.

컴퓨터에서 컴퓨터로

  1. 두 컴퓨터를 동일한 WiFi에 연결하고 8ma Quick Transfer를 여세요.
  2. 보내는 쪽에서 파일을 선택하고 동일 WiFi 공유를 켜세요.
  3. 받는 컴퓨터에서 주변 파일을 열고 다운로드를 확인하세요.

컴퓨터에서 휴대폰으로

단계는 동일합니다. 컴퓨터에서 파일을 선택하고 Nearby Sharing을 켠 다음, 휴대폰에서 사이트를 여세요. 휴대폰에서 컴퓨터로 다시 보내는 것도 같은 방식으로 작동합니다.

왜 편리한가요?

  • 복사할 긴 링크가 없고 로그인할 공유 계정도 없습니다.
  • Windows, macOS, Android, iPhone 모두 브라우저를 사용할 수 있습니다.
  • 파일이 이동하는 동안 속도와 남은 시간이 계속 표시됩니다.

공용 WiFi를 사용해도 되나요?

대신 링크, QR 코드 또는 픽업 코드를 사용하세요. 공용 네트워크에서는 Nearby Sharing을 끄면 다른 기기가 공유로 표시된 파일 이름을 볼 수 없습니다.
