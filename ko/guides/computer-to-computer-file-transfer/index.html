컴퓨터 전송

컴퓨터에서 컴퓨터로 파일을 어떻게 전송하나요?

USB 드라이브, 케이블 또는 공유 폴더가 없어도 두 컴퓨터는 브라우저를 통해 전송을 시작할 수 있습니다. 설치할 앱도, 만들 계정도 필요 없습니다.

간단한 답변

양쪽 컴퓨터에서 8ma Quick Transfer를 여세요. 보내는 쪽에서 파일을 선택하고, 동일한 WiFi에서는 근처 공유를 활성화하세요. 다른 네트워크에서는 링크나 픽업 코드를 대신 보내세요.

동일한 WiFi에서 전송하기

  1. 양쪽 컴퓨터에서 8ma Quick Transfer를 여세요.
  2. 보내는 쪽에서 파일을 선택하고 동일 WiFi 공유를 활성화하세요.
  3. 받는 컴퓨터의 '주변 파일'에서 해당 파일을 선택하고 다운로드를 확인하세요.

컴퓨터가 서로 다른 네트워크를 사용한다면 어떻게 하나요?

보내는 사람이 임시 링크를 복사하거나, QR 코드를 표시하거나, 8자리 픽업 코드를 공유할 수 있습니다. 받는 사람은 링크를 열거나 코드를 입력하여 다운로드합니다.

Windows와 Mac 간에 파일을 서로 전송할 수 있나요?

네. 최신 브라우저를 통해 Windows, macOS, Linux 컴퓨터 간에 파일을 주고받을 수 있습니다. 매우 큰 파일의 경우, 최신 데스크톱 Chrome 또는 Edge를 사용하세요.

대용량 파일에도 작동하나요?

네. 별도의 클라우드 업로드가 없으며, 전송은 사용 가능한 네트워크 대역폭을 사용합니다. 데스크톱 Chrome과 Edge는 최대 100GB 파일을 지원합니다.
