휴대폰과 컴퓨터 전송

휴대폰과 컴퓨터 간에 파일을 어떻게 전송하나요?

휴대폰과 컴퓨터에서 8ma Quick Transfer를 열어 사진, 동영상, 문서 또는 압축 파일을 보내세요. 찾아야 할 케이블, 설치할 앱, 로그인할 공유 계정이 없습니다.

간단한 답변

보내는 기기에서 파일을 선택하세요. 두 기기가 같은 WiFi에 있을 때 Nearby Sharing을 사용하거나, 다른 곳에서는 링크, QR 코드 또는 8자리 픽업 코드를 보내세요.

같은 WiFi에서 전송하기

  1. 휴대폰과 컴퓨터를 같은 WiFi에 연결한 후, 두 기기에서 8ma Quick Transfer를 여세요.
  2. 보내는 기기에서 파일을 선택하고 동일 WiFi 공유를 켜세요.
  3. 받는 기기에서 Nearby Files 아래의 파일을 선택하고 다운로드를 확인하세요.

기기가 서로 다른 네트워크를 사용한다면 어떻게 하나요?

파일을 선택한 후 생성된 링크, QR 코드 또는 픽업 코드를 받는 사람에게 보내세요. 받는 사람이 파일을 확인하고 다운로드하는 동안 두 페이지를 모두 열어 두세요.

아이폰과 안드로이드에서 작동하나요?

네. Windows, macOS, 아이폰 및 안드로이드 기기는 최신 브라우저를 사용할 수 있습니다. 휴대폰에서 컴퓨터로 다시 보낼 때도 동일한 단계가 적용됩니다.

대용량 파일을 위해 무엇을 확인해야 하나요?

실제 속도는 두 기기와 사용 가능한 네트워크 대역폭에 따라 달라집니다. 두 페이지를 열어 두고, 휴대폰이 잠기지 않도록 하며, 컴퓨터가 절전 모드로 전환되지 않도록 유지하세요. 매우 큰 파일의 경우 최신 데스크톱 Chrome 또는 Edge를 권장합니다.
