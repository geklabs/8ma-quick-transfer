Privacy

Updated July 23, 2026

What this promotion site collects

This site has no user accounts, forms, advertisements, or behavior analytics. It does not save file contents in the browser. To remember your language choice, it stores one local value named 8ma-promo-language.

Hosting logs

This promotion site is hosted by GitHub Pages. GitHub may log technical information needed to operate the service, such as IP address, browser type, and request time, according to its own policies. This project cannot access or control those infrastructure logs.

Promotion site and product

The “Send a file” action opens t.8ma.co. The product site's live connections and file transfers are outside this promotion page.
