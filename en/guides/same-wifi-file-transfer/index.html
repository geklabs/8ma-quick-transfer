Same WiFi transfer

Open the site on both devices and start sharing.

On the same WiFi, computer to computer and computer to phone transfers need no extra app or shared account. Open 8ma Quick Transfer on both devices, then enable Nearby Sharing on the sender.

The simplest route

The sender chooses a file and turns on “Same WiFi sharing.” The receiver opens the site on the other device and selects the file under Nearby Files.

Computer to computer

  1. Connect both computers to the same WiFi and open 8ma Quick Transfer.
  2. Choose the file on the sender and turn on Same WiFi sharing.
  3. Open the nearby file on the receiving computer and confirm the download.

Computer to phone

The steps are the same. Choose the file on the computer, turn on Nearby Sharing, and open the site on the phone. Sending from a phone back to a computer works the same way.

Why is it convenient?

  • No long link to copy and no shared account to sign into.
  • Windows, macOS, Android, and iPhone can use a browser.
  • Speed and time remaining stay visible while the file moves.

Can I use public WiFi?

Use a link, QR code, or pickup code instead. Keep Nearby Sharing off on public networks so other devices do not see the filename shown by the share.
