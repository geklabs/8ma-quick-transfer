Transferencia entre computadoras

¿Cómo transfieres archivos de computadora a computadora?

Incluso sin una memoria USB, cable o carpeta compartida, dos computadoras pueden comenzar a transferir a través de sus navegadores. No hay aplicación que instalar ni cuenta que crear.

La respuesta corta

Abre 8ma Quick Transfer en ambas computadoras. Elige el archivo en el emisor; en la misma WiFi, activa el uso compartido cercano. En redes diferentes, envía un enlace o código de recogida en su lugar.

Transfiere en la misma WiFi

  1. Abre 8ma Quick Transfer en ambas computadoras.
  2. Elige el archivo en el emisor y activa el uso compartido en la misma WiFi.
  3. Selecciona el archivo en Archivos cercanos en la computadora receptora y confirma la descarga.

¿Qué pasa si las computadoras usan redes diferentes?

El emisor puede copiar un enlace temporal, mostrar un código QR o compartir el código de recogida de 8 caracteres. El receptor abre el enlace o ingresa el código para descargar.

¿Pueden Windows y Mac transferirse archivos entre sí?

Sí. Un navegador actual permite que computadoras con Windows, macOS y Linux intercambien archivos. Para archivos muy grandes, usa la versión de escritorio actual de Chrome o Edge.

¿Funciona esto para archivos grandes?

Sí. No hay una subida separada a la nube y la transferencia usa el ancho de banda de red disponible. Chrome y Edge de escritorio admiten archivos de hasta 100 GB.
